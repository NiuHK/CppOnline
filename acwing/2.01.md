i 物品

j  背包容量

先i后j   判断第i个物品下，若空为j，找最大

| i\j | 0 | 1 | 2 | 3 |
| --- | - | - | - | - |
| 0   |   |   |   |   |
| 1   |   |   |   |   |
| 2   |   |   |   |   |
| 3   |   |   |   |   |


if(j<v[i])

s[i][j] =s[i-1][j];

else

s[i][j]=max(s[i-1][j],s[i-1][j-v[i]]+w[i]);

可知，s[i][]是由s[i-1]所得，且题目只求最终s[n][m]。

故，可以优化为一维，又因 `s[i][]是由s[i-1]所得`，故应倒序遍历。
